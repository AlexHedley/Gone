# Gone

Gone is the GO compiler for .NET.

Come watch it be built: https://twitch.tv/FrankKrueger

## Building

```
dotnet build
```

### Modifying the Grammar

If you modify Parser.jay, then please rebuild the parser with:

```
make
```
